# Wedding Website

A static wedding site with a photo gallery.

## Adding Photos to the Gallery

1. **Add image files** to `assets/images/` (supports jpg, jpeg, png, webp, gif).
2. **Push to GitHub** — a workflow runs automatically, updates the gallery, and pushes the change. Works locally (open the HTML file) and on GitHub Pages.
3. **Local preview first:** Run `node scripts/build-gallery.js` to see new images before pushing.
